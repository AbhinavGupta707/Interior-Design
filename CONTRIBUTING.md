# Contributing

Implementation is checkpointed. Before editing, read the active M1 plan, root `AGENTS.md`, the active checkpoint contract and the orchestration ledger.

Use conventional, scoped commits. Never commit secrets, customer data, captures, generated renders, provider payloads, datasets or model weights. New external data/code/assets require a source, licence, allowed-purpose and version record.

Every change must include proportionate tests and an honest handoff. Shared contract/root changes are orchestrator-owned and must be committed before dependent worktree lanes start.
