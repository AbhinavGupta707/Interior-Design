# Synthetic C8 fixture

`synthetic-appearance-gsplat.json` and the literal frame bytes used by its
tests are created data, public-domain for this repository, and contain no
customer, provider, property, face, address, credential, path, object key or
signed URL. Service processing is allowed only for the fixture tests and
training use is explicitly denied. The fixture validates adapter contracts;
it is not Nerfstudio, gsplat, CUDA, accuracy or production-capacity evidence.
